# SHIKOR (শিকড়) 🇧🇩
### Digital Archive of Bangladesh's Indigenous Heritage

**SHIKOR** is a trilingual web application (Kudmali, Bangla, English) dedicated to preserving the history, lifestyle, and culture of the Mahato and other indigenous communities of Bangladesh.

## 📱 Features
- **Lifestyle Comparison:** Explore the evolution of Mahato lifestyle (Old vs New).
- **Trilingual Support:** Switch between English, Bangla, and Kudmali.
- **Cultural Hub:** Information on the Holy Ainash and traditional festivals.
